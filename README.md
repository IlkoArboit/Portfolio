# Portfolio – Ilko Arboit

Este repositorio contiene mi portfolio profesional como desarrollador de software full stack. El objetivo del proyecto es presentar de forma clara mis habilidades técnicas, experiencia y algunos de los proyectos en los que he trabajado.

## Sobre mí
Soy desarrollador full stack con más de tres años de experiencia en el desarrollo de aplicaciones web y móviles. Me especializo en backend con Golang, Python y Express, y en frontend con React, Svelte y React Native. Tengo experiencia en diseño de APIs REST, automatización de procesos e integración de sistemas.

## Tecnologías y herramientas
- **Frontend:** React, Svelte, React Native  
- **Backend:** Golang, Python, Express  
- **Bases de datos:** MySQL, PostgreSQL  
- **Automatización e integraciones:** n8n, Webhooks, APIs REST  
- **Herramientas:** Git, Docker, Postman, Linux, VS Code  

## Objetivo del portfolio
Mostrar mi perfil técnico, mi forma de trabajar y mi enfoque en la creación de soluciones escalables, mantenibles y orientadas a resolver problemas reales. Este portfolio está pensado tanto para reclutadores como para otros desarrolladores interesados en mi trabajo.
